# DeepFake Detector

A simple web application that detects whether an uploaded image is **Real** or **DeepFake** using a model trained with **Google Teachable Machine**.

## Features

* Detects Real vs DeepFake images
* Clean and responsive interface
* Built with HTML, CSS, and JavaScript

## Tech Stack

* HTML
* CSS
* JavaScript
* Google Teachable Machine

## Live Demo

**GitHub Pages:** https://sahil-pasalkar101.github.io/Deepfake-Detection-Web-Application/

## How to Use

1. Click the **Start** button.
2. Upload an image.
3. Wait a few seconds while the model analyzes the image.
4. View the prediction (Real or DeepFake).

## Getting Started

1. Clone this repository.
2. Open `index.html` in your browser.

## Note

This project is for educational purposes. The accuracy of predictions depends on the quality of the trained model and dataset.
